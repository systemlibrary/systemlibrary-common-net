## Description
A library of classes and methods for any .NET >= 5 application

Selling points:
- Common extension on strings, enums, arrays...
- Create your own json file and inherit Config&lt;&gt; class and your json is auto-read into a C# class
- Simple "console.log" through Dump.Write()
- Simple extension method on object and string named "".ToJson();
- Simple extension method for converting parts of a json through "".PartialJson();
- Simple "Fire and Forget" through Async.Run()

## Requirements
- &gt;= .NET 5
