# SystemLibrary Common Net

## Requirements
- .NET 4.8

## Latest Version
- Extended IEnumerableExtensions with Has() overload
- Added UnitTests for IEnumerableExtensions.Has() method

## Description
- Common extension methods for strings, objects, uris, etc...
- Contains a Config<T> class to easily create custom configuration files, which you declare in web or app.config
- See the 'docs' section to read more and see example code

## Install
- Install through Nuget Package Manager in Visual Studio
