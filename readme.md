# SystemLibrary Common Net

## Requirements
- &gt;= .NET 5

## Latest Version
- Exception occured in Dump.Write if not systemLibraryCommonNet.Dump existed in appSettings
- Added unit tests

## Description
Base library for .NET Appplications
- Common extension on strings, enums, arrays...
- Create your own json file and inherit Config&lt;&gt; class and your json is auto-read into the C# class
- Simple "console.log" through Dump.Write()
- Simple extension method on object and string named 'ToJson()'
- Simple "Fire and Forget" through Async.Run()
